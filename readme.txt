mailgo, a different mailto

== Description ==

mailgo [https://mailgo.js.org](https://mailgo.js.org) for WordPress

**This plugin is WIP!**

Instead of the classic behaviour of mailto links, mailgo will show a modal with more possibilities (like, for example, to copy the email address...)

== Installation ==

1. Extract and upload `mailgo.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
